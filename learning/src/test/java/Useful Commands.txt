Register nodes in selenium Ubuntu with chromedriver

java -Dwebdriver.chrome.driver=/home/arvinder/Downloads/chromedriver -jar selenium-server-standalone-3.4.0.jar -role node -hub http://172.17.0.1:4444/grid/register -browser "browserName=chrome, version=ANY, maxInstances=10, platform=LINUX"

Start selenium Hub (Simple command)

java -jar selenium-server-standalone-3.4.0.jar -role hub
